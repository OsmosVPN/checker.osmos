# OSMOS Checker

Приложение для проверки доступности конфигураций.

## О конфигурациях

**Важно:** Все конфигурации в этом приложении взяты и агрегированы из открытых источников (GitHub репозитории). Приложение не создает и не хранит собственные конфигурации, а только загружает и проверяет публично доступные списки.

Источники конфигураций:

- [etoneyaproject.github.io](https://github.com/EtoNeYaProject/etoneyaproject.github.io)
- [zieng2/wl](https://github.com/zieng2/wl)
- [igareck/vpn-configs-for-russia](https://github.com/igareck/vpn-configs-for-russia)
- [ByeWhiteLists/ByeWhiteLists2](https://github.com/ByeWhiteLists/ByeWhiteLists2)
- [RKPchannel/RKP_bypass_configs](https://github.com/RKPchannel/RKP_bypass_configs)
